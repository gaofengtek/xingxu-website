# 星旭官网全站优化设计文档

**日期：** 2026-05-28  
**状态：** 已确认  
**基于：** [2026-05-26 初始设计](2026-05-26-xingxu-website-design.md)

## 1. 设计方向

- **风格：** 现代科技感 + 企业大气感
- **Hero：** 保留视频，叠加玻璃拟态卡片 + 标语轮播
- **范围：** 全站 5 页面优化

## 2. 首页重构（index.html）

### 2.1 Hero 区域
- 视频背景保留（hero.mp4）
- 中央玻璃拟态卡片（backdrop-filter: blur + 半透明背景）
- 3 条标语轮播（CSS animation）
- 渐变遮罩优化（深蓝→透明）

### 2.2 新增：场景化解决方案
- 三栏布局：高校教学 / 科研院所 / 能源企业
- 每栏顶部色条（深蓝/金橙/浅蓝）+ 图标 + 场景描述

### 2.3 新增：数据亮点
- 深蓝背景 + 金橙数字
- 四项数据：7+领域 / 10年经验 / 100+模型 / 3大业务
- 滚动到可见时数字增长动画

### 2.4 核心业务卡片（保留，优化）
- 增加 hover 上浮效果 + 玻璃拟态

### 2.5 覆盖领域标签（保留，优化）

## 3. 子页面视觉统一

- 统一 hero-sub 样式（深蓝渐变 + 图标装饰）
- 统一卡片体系（白底 + 细边框 + 统一阴影）
- 统一间距节奏（section 60px, 卡片间距 24px）

## 4. 动效交互

| 动效 | 应用位置 | 实现方式 |
|------|---------|---------|
| 导航透明→实色 | 全局导航栏 | scroll 事件 + CSS transition |
| 滚动揭示 | 首页各区块 | IntersectionObserver + CSS |
| 卡片悬浮 | 所有卡片 | CSS transform + box-shadow |
| 数字增长 | 数据亮点区 | IntersectionObserver + JS animate |

## 5. 导航与页脚

### 5.1 导航栏
- Hero 区透明背景 + 白色文字
- 滚动超过 Hero 后切换白底 + 阴影 + 深色文字
- 平滑过渡动画

### 5.2 页脚
- 深蓝背景 + 四栏布局
- 公司信息 / 快速链接 / 联系方式 / 备案信息

## 6. 技术边界

- 纯静态 HTML/CSS/JS，不引入框架
- 不引入 Three.js（Hero 用视频 + CSS 实现）
- 不引入额外依赖
- 响应式（PC + 手机）
- 性能：首次内容绘制 < 1.5s
